\# OK42 3D AR 模板



这是 OK42 的 3D AR 模板仓库，用于在 GitHub Pages 上展示 3D 模型和 AR 功能。



\## 使用方法



1\. 打开 GitHub Pages：

&nbsp;  https://ok42com-pixel.github.io/ok42-3d-template/

2\. 页面会自动加载默认 3D 模型。

3\. 替换模型：

&nbsp;  - 上传新的 `.glb` 或 `.usdz` 文件到 `/models/`

&nbsp;  - 上传新的首帧图到 `/poster/`

&nbsp;  - 修改 `index.html` 中的 `src` 和 `poster` 路径

4\. 页面会自动显示新模型。

5\. 生成更多 3D 页面：

&nbsp;  - 复制 `index.html` → 改名为 `page2.html`、`page3.html` 等  

&nbsp;  - 修改页面里的 `src`、`poster` 对应新的模型  

&nbsp;  - 这样仓库里就可以管理多个 3D 页面
